jQuery plugin transforming Unsplash images into puzzles

Scrambler

This jQuery plugin uses random Unsplash images to create a sliding image puzzle, Scrambler. You can specify square size and add it to any element, class or ID on your site. The plugin sources Unsplash images using Unsplash.it.

Download on Github

Unsplash

Unsplash is a side project created by Crew, a Montreal-based online marketplace for creative talent. They provide free "do whatever you want" high-resolution photos.

Check out Unsplash

How to use Scrambler

  1. Find the scrambler.js file.
  2. Insert the ID you want your puzzle to show up in and square size into this line:
    $('#scrambler').scrambler(80);
  3. Choose different square sizes based on your media queries: 
    if (window_width < 520) {
  $('#scrambler').scrambler(80);
} if ((window_width > 520) && (window_width < 800)){
  $('#scrambler').scrambler(130); 
} if (window_width > 800){
  $('#scrambler').scrambler(150); 
}
